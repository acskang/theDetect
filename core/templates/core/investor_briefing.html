OBJECT DETECTION AUTOMATION

제품 이미지 데이터를 현장용 AI 모델로 자동 전환해, 새로운 탐지 서비스를 빠르게 출시할 수 있습니다.

theDetect는 탐지할 물체를 정의하고, 이미지를 등록·라벨링·보강·학습한 뒤, 스마트폰에서 바로 물체를 찾을 수 있게 만드는 서비스입니다.

기존 작업 방식과 theDetect 방식

흩어진 작업 방식

  • 목표 물체 목록을 따로 정리합니다.
  • 이미지와 표시 작업을 다른 도구에서 관리합니다.
  • 학습 결과를 스마트폰용으로 옮기는 과정이 끊깁니다.
  • 현장 테스트와 기록 확인이 분리됩니다.

theDetect 방식

  • 목표 정의부터 스마트폰 배포까지 한 화면 흐름으로 이어집니다.
  • 이미지 등록, 라벨링, 보강, 학습 데이터 생성을 같은 서비스에서 진행합니다.
  • 서버에서 먼저 확인한 뒤 스마트폰 모델로 내보냅니다.
  • 스마트폰이 현장에서 자체 탐지를 수행하고 기록을 남깁니다.

투자자가 확인할 포인트

반복 가능한 제작 절차

새 물체를 추가해도 같은 순서로 이미지와 모델을 만들 수 있습니다.

스마트폰 중심의 현장성

서버 연결이 불안정해도 다운로드된 모델로 현장에서 탐지할 수 있습니다.

운영 기록 축적

테스트와 탐지 기록을 남겨 다음 개선 작업의 근거로 활용합니다.

시연에서는 전체 자동화 흐름을 순서대로 확인합니다.

클래스 정의에서 시작해 스마트폰 자체 탐지까지 이어지는 절차가 이 서비스의 핵심입니다. 먼저 시연 절차를 보고, 실제 화면에서 각 단계를 확인합니다.
